Teaching Assistant / Tutor for Python Programming for Data Scientists course at the GippLab department.

Role and Responsibilities
======

As a teaching assistant for this graduate-level course, I provided comprehensive support to students learning Python for data science applications. My responsibilities included:

* Supporting students during class sessions and office hours
* Helping students debug code and understand programming concepts
* Guiding students through hands-on data science projects

Research Support
======

In addition to teaching duties, I contributed to research activities:

* **Web Scraping**: Developed web scrapers to collect data from various online sources
* **NLP Dataset Curation**: Built and curated a specialized NLP dataset for news media bias research
* **Data Processing**: Assisted students with data cleaning and preprocessing techniques

Skills Developed
======

This teaching experience enhanced my abilities in:

* Explaining complex programming concepts to diverse audiences
* Debugging and troubleshooting Python code
* Web scraping and data collection methodologies
* Natural Language Processing (NLP) dataset preparation
* Supporting research projects in computational linguistics
